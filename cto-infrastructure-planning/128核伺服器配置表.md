## 五、技術規格詳細說明

### 5.1 CPU規格（AMD EPYC 7713）

```
核心架構：
├─ 代號：Milan（第3代EPYC）
├─ 製程：7nm
├─ 核心數：64核/128線程（單顆）
├─ 基礎時脈：2.0 GHz
├─ 加速時脈：最高3.675 GHz
├─ L3快取：256MB（共享）
├─ TDP：225W
└─ 支援：PCIe 4.0、DDR4-3200 8通道

特點：
✅ 多核性能優異（適合容器化）
✅ 大容量L3快取（減少記憶體訪問）
✅ 8通道記憶體（頻寬高）
✅ PCIe 4.0（I/O快）
✅ 性價比高（比Intel便宜20%+）
```

### 5.2 記憶體配置（300GB ECC）

```
配置方式1：12條×25GB（無預留）
├─ 每通道：1.5條
├─ 雙面DIMM
└─ 無擴充空間

配置方式2：10條×32GB - 2條預留（推薦）✅
├─ 每通道：1.25條
├─ 雙面DIMM
├─ 保留2個插槽
└─ 可擴充至384GB

特點：
✅ ECC糾錯（企業級必備）
✅ DDR4-3200（高頻寬）
✅ RDIMM（註冊記憶體，穩定性高）
```

### 5.3 儲存配置（RAID6，實際8TB）

```
配置：10× 1TB SATA SSD RAID6
├─ 裸容量：10TB
├─ RAID6雙校驗：8TB可用
├─ 允許2顆硬碟同時故障
└─ 適合大容量需求

vs RAID10（4× 1.92TB）：
- RAID10：實際3.84TB，快但貴
- RAID6：實際8TB，較慢但便宜
- 選擇：RAID6（CP值高）✅

寫入速度：
- RAID10：約800MB/s
- RAID6：約600MB/s
- 差異：25%，但容量多2倍
```

### 5.4 網路配置（雙25GbE）

```
配置：雙埠25GbE SFP28
├─ 每埠：25 Gbps
├─ 總頻寬：50 Gbps（全雙工）
├─ 網路冗餘：HA
└─ 含SFP+模組

vs 雙10GbE：
- 10GbE：20 Gbps總頻寬
- 25GbE：50 Gbps總頻寬（+150%）
- 成本差異：約+5000元
- 建議：選25GbE ✅
```
