## 九、重要提醒

### ⚠️ 採購注意事項

```
1. 確認CPU型號
   ✅ 一定要AMD EPYC 7713（64核×2=128核）
   ❌ 不要接受 EPYC 7643（48核×2=96核）
   ❌ 除非價格差很多且您願意接受96核

2. 確認是否含機架
   ✅ 要求報價含42U機櫃
   ✅ 確認機櫃品質（不要太陽春）

3. 確認安裝服務
   ✅ 含到府安裝
   ✅ 含基礎設定（RAID、BIOS）
   ✅ 含上架與佈線

4. 確認交貨時間
   ✅ 1/20前到貨
   ✅ 確認是「到貨」還是「出貨」
   ✅ 要求寫進合約

5. 確認保固內容
   ✅ 3年7×24次日到府
   ✅ 確認保固起算日
   ✅ 確認備品提供

6. 確認付款條件
   ✅ 爭取貨到付款或月結
   ✅ 至少爭取30天帳期
```

### 💰 議價技巧

```
1. 同時多家詢價
   - 讓廠商知道有競爭壓力

2. 強調後續採購
   - 「這是第一期，後續還有2-4台」

3. 強調預算限制
   - 「我們是新創公司，預算有限」

4. 要求整體優惠
   - 「包含機櫃和安裝，80萬以內」

5. 適時施壓
   - 「其他家報XX萬，你們可以優惠嗎？」

6. 保持彈性
   - 「如果價格好，我可以馬上簽約」
```

## 十、常見問題 FAQ

**Q1：哪家最推薦？**
A1：精誠資訊，服務最完整，價格合理。

**Q2：一定要同時詢價嗎？**
A2：是的，這樣才有議價空間，也可以比較服務。

**Q3：需要準備什麼資料？**
A3：公司名稱、統編、聯絡人、Email、預算、交貨地址。

**Q4：可以分期付款嗎？**
A4：可以，通常SI都有企業分期方案。

**Q5：保固真的有用嗎？**
A5：非常有用！伺服器故障時7×24次日到府很重要。

**Q6：一定要買機櫃嗎？**
A6：是的，伺服器需要專用機櫃，不能隨便放。

**Q7：可以先看機器嗎？**
A7：可以要求廠商提供Demo或到展示中心參觀。

**Q8：報價後還能議價嗎？**
A8：可以！第一次報價通常還有議價空間。

**Q9：128核會不會太多？**
A9：不會，您有3個專案要跑，128核才夠用。

**Q10：為什麼選AMD不選Intel？**
A10：AMD EPYC性價比更高，核心數多，多核性能優異。
